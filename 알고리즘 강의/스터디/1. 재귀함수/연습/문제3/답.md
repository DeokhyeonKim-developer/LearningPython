### 3\. 피보나치 수열 (Fibonacci Sequence) 정답

```python
def fibonacci(n):
  # Base Case 1 & 2: n이 0이면 0을, 1이면 1을 반환합니다.
  if n <= 1:
    return n
  # Recursive Case: n > 1 이면 (n-1)번째 항과 (n-2)번째 항의 합을 호출합니다.
  else:
    return fibonacci(n - 1) + fibonacci(n - 2)

# 테스트
print(f"피보나치 6번째 항 = {fibonacci(6)}") # 8 (0, 1, 1, 2, 3, 5, 8)
print(f"피보나치 0번째 항 = {fibonacci(0)}") # 0
print(f"피보나치 1번째 항 = {fibonacci(1)}") # 1
```

**설명:** `fibonacci(6)`은 `fibonacci(5) + fibonacci(4)`를 호출합니다. 이 과정이 `n`이 0 또는 1이 될 때까지(Base Cases) 계속해서 두 갈래로 나뉘며 반복됩니다.
