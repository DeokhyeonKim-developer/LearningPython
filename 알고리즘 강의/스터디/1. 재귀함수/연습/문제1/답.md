### 1\. 팩토리얼 (Factorial) 정답

```python
def factorial(n):
  # Base Case (기본 단계): n이 0이면 1을 반환하며 재귀를 멈춥니다.
  if n == 0:
    return 1
  # Recursive Case (재귀 단계): n > 0 이면 n * (n-1)!을 재귀적으로 호출합니다.
  else:
    return n * factorial(n - 1)

# 테스트
print(f"5! = {factorial(5)}") # 120
print(f"0! = {factorial(0)}") # 1
```

**설명:** `factorial(5)`는 `5 * factorial(4)`를, `factorial(4)`는 `4 * factorial(3)`을 호출... 마지막으로 `factorial(0)`이 1을 반환할 때까지 재귀가 이어집니다.
