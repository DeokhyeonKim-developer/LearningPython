### 2\. 리스트의 합 (Sum of List) 정답

```python
def list_sum(nums):
  # Base Case: 리스트가 비어있으면(길이가 0이면) 0을 반환합니다.
  if not nums: # len(nums) == 0 과 동일합니다.
    return 0
  # Recursive Case: 리스트의 첫 번째 요소(nums[0])와
  # 나머지 리스트(nums[1:])의 합을 재귀적으로 더합니다.
  else:
    return nums[0] + list_sum(nums[1:])

# 테스트
print(f"[1, 2, 3, 4, 5]의 합 = {list_sum([1, 2, 3, 4, 5])}") # 15
print(f"[]의 합 = {list_sum([])}") # 0
```

**설명:** `list_sum([1, 2, 3])`은 `1 + list_sum([2, 3])`을, `list_sum([2, 3])`은 `2 + list_sum([3])`을... 마지막으로 `list_sum([])`이 0을 반환할 때까지 재귀가 이어집니다.
