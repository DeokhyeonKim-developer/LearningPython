### 2\. 특정 합계를 만족하는 조합 찾기

**목표:** `nums` 리스트에서 `k`개를 뽑은 조합 중, 합이 `target`이 되는 조합을 찾습니다.

**핵심 로직:**
1번 문제에서 만든 `find_combinations` 함수를 그대로 사용하여 `k`개의 모든 조합을 구한 뒤, 리스트 컴프리헨션(List Comprehension)이나 `for`문을 사용해 합(sum)이 `target`과 일치하는 조합만 필터링합니다.

**코드:**

```python
# (1번 문제의 find_combinations 함수가 필요합니다)
def find_combinations(arr, k):
    results = []
    if k == 0:
        return [[]]
    if not arr or k > len(arr):
        return []
    
    first = arr[0]
    rest = arr[1:]
    
    combinations_with_first = find_combinations(rest, k - 1)
    for combo in combinations_with_first:
        results.append([first] + combo)
        
    combinations_without_first = find_combinations(rest, k)
    results.extend(combinations_without_first)
    
    return results

def find_sum_combinations(nums, k, target):
    """
    nums에서 k개를 뽑은 조합 중 합이 target이 되는 조합을 찾습니다.
    """
    # 1. k개의 모든 조합을 구합니다.
    all_combos = find_combinations(nums, k)
    
    # 2. 합(sum)이 target과 일치하는 조합만 필터링합니다.
    result = [combo for combo in all_combos if sum(combo) == target]
    
    return result

# --- 실행 ---
nums = [2, 4, 6, 8, 10]
k = 3
target = 18
print(f"입력: nums={nums}, k={k}, target={target}")
print("출력:", find_sum_combinations(nums, k, target))
```

**실행 결과:**

```
입력: nums=[2, 4, 6, 8, 10], k=3, target=18
출력: [[2, 6, 10], [4, 6, 8]]
```
