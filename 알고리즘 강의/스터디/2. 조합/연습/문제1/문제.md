### 문제 1: nC\_k 조합 구하기

**목표:** 주어진 리스트에서 `k`개의 원소를 뽑아 만들 수 있는 모든 고유한 조합을 반환하는 함수를 작성하세요.

**설명:**
가장 기본적인 조합 문제입니다. 예를 들어 `[1, 2, 3, 4]`에서 2개를 뽑는 조합은 `[1, 2]`, `[1, 3]`, `[1, 4]`, `[2, 3]`, `[2, 4]`, `[3, 4]`입니다. 순서가 다르더라도 구성이 같으면(예: `[1, 2]`와 `[2, 1]`) 하나의 조합으로 봅니다. 이 문제를 해결하는 함수 `find_combinations(arr, k)`를 구현해 보세요.

**입력 예시:**

  * `arr = [1, 2, 3, 4]`
  * `k = 2`

**출력 예시:**

```
[[1, 2], [1, 3], [1, 4], [2, 3], [2, 4], [3, 4]]
```

*(출력 순서는 다를 수 있습니다.)*
