### 3\. 모든 크기의 조합 구하기 (부분 집합)

**목표:** 주어진 리스트 `items`로 만들 수 있는 **비어있지 않은** 모든 조합(부분 집합)을 구합니다.

**핵심 로직:**
1번 문제의 `find_combinations` 함수를 재사용합니다. 뽑는 개수 `k`를 1개부터 리스트의 전체 길이(`len(items)`)까지 `for`문으로 반복하면서, 각 `k`값에 대한 조합을 모두 구해 하나의 리스트에 합칩니다.

**코드:**

```python
# (1번 문제의 find_combinations 함수가 필요합니다)
def find_combinations(arr, k):
    results = []
    if k == 0:
        return [[]]
    if not arr or k > len(arr):
        return []
    
    first = arr[0]
    rest = arr[1:]
    
    combinations_with_first = find_combinations(rest, k - 1)
    for combo in combinations_with_first:
        results.append([first] + combo)
        
    combinations_without_first = find_combinations(rest, k)
    results.extend(combinations_without_first)
    
    return results

def find_all_subsets(items):
    """
    주어진 리스트로 만들 수 있는 비어있지 않은 모든 조합(부분 집합)을 구합니다.
    """
    all_subsets = []
    n = len(items)
    
    # k=1 (1개 뽑는 조합) 부터 k=n (n개 뽑는 조합) 까지 반복
    for k in range(1, n + 1):
        combos_of_size_k = find_combinations(items, k)
        all_subsets.extend(combos_of_size_k) # 결과 리스트에 추가
        
    return all_subsets

# --- 실행 ---
items = ['A', 'B', 'C']
print(f"입력: items={items}")
print("출력:", find_all_subsets(items))
```

**실행 결과:**

```
입력: items=['A', 'B', 'C']
출력: [['A'], ['B'], ['C'], ['A', 'B'], ['A', 'C'], ['B', 'C'], ['A', 'B', 'C']]
```
